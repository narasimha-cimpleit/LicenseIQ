üîê Navigation Permission System

Menu visibility is determined by a 3-tier priority system. The system checks in order until it finds a match.

1
User Override (Highest Priority)

Individual user-specific setting. If exists, use this and stop checking.

2
Enabled Toggle (Role Permission)

Role-based setting from Configuration page. ON = Show, OFF = Hide.

3
Default Access (Baseline)

Uses defaultRoles from seed data if no explicit permission was set.

Permission Decision Example

Menu Item Default Access Enabled Toggle User Sees Menu? Why?

Contracts ON OFF NO Enabled Toggle overrides
Calculations OFF ON YES Enabled Toggle overrides
User Management OFF Not Set NO Uses Default Access
